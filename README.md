# The bulan.io Backend Server

## Installation

- First, install Rust and Cargo. See <https://rustup.rs/>.
- Next, clone this repository.

  ```console
  git clone https://github.com/bulan-io/bulan-server.git
  ```

- Then, build the project.

  ```console
  cargo build --release
  ```

## Usage

- Run the server.

  ```console
  cargo run --release
  ```
